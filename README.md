# Frequency Analysis of Words

This programm help to find the more popular words in the file. 

#How to use 

In terminal: 
<pre>
    <code>
    $python3 --path "filepath"
    </code>
</pre>

#Example

<pre>
    <code>
        $python3 --path "/Users/user/text.txt"
    </code>
</pre>

<pre>
    <code>
        word_1 -> 5
        word_2 -> 2
        word_3 -> 2
        word_4 -> 1
    </code>
</pre>
